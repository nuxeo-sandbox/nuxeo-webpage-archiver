# nuxeo-webpage-archiver

_()Work in Progress)_

### Build

    cd /path/to/nuxeo-webpage-archiver
    mvn clean install    

*NOTE* As of today, unit test are OK in Eclipse, not in Maven. So better run:

    cd /path/to/nuxeo-webpage-archiver
    mvn clean install -DskipTests
